# ConsultMAI

**Turn complexity into execution clarity.**

Most business problems aren't unsolvable — they're unstructured.  
ConsultMAI takes a messy situation and converts it into a structured  
execution roadmap in seconds.

Not a chatbot. Not a Q&A tool. A thinking framework, powered by AI.

---

## The Problem It Solves

When you're inside a business problem — revenue dropping, operations breaking,  
team falling apart — it's hard to think clearly. You know *something* is wrong  
but the path from "mess" to "action" is unclear.

Consultants charge ₹5–50L to structure that thinking.  
ConsultMAI does it instantly.

---

## What You Get

Input a situation. Get back a structured brief:

| Section | What it contains |
|---|---|
| **Situation** | Reframed problem statement |
| **Priority Flow** | Sequenced decision path |
| **Key Components** | MECE breakdown of the problem |
| **Execution Steps** | Ordered actions with context |
| **Notes / Risks** | What could go wrong and what to watch |

Three output modes: **Summary**, **Executive**, **Detailed**  
Export to Markdown for reports, decks, or client briefs.

---

## Who It's For

- Consultants and freelancers structuring client problems
- CA / Legal professionals navigating complex situations
- Founders diagnosing growth or ops breakdowns
- Anyone who needs structured thinking fast

## Getting Started

```bash
git clone https://github.com/your-username/consultmai.git
cd consultmai
pip install -r requirements.txt
```

Create a `.env` file:
```
ANTHROPIC_API_KEY=your_key_here
USE_REAL_API=true
```

Run:
```bash
streamlit run app.py
```
